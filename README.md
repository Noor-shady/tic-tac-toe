# Tic Tac Toe Project

"This is a classic game built with web code."

## Table of Contents
- [Project Files](#project-files)
- [How to Run](#how-to-run)
- [How to Play](#how-to-play)

## Project Files
- **`index.html`** — The main board.
- **`style.css`** — The colors and grid.
- **`script.js`** — The game rules.

## How to Run
Right-click on index.html then open with live Server then later on you can play it.

## How to Play
- Player **"X"** goes first then **"O"**.  
- Click any box to make a move.  
- Get **3 in a row** to win.
